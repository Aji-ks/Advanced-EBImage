Menmpilkan dengan Method=“browser”

dengan method browser ini akan membuat gambar bisa di zoom in dan di zoom out. pada penulisan profil.jpg harus terdapat gambar tersebut pada project kita atau sesuai dengan work directory

img = readImage("profil.jpg")
display(img, method="browser")

Menambahkan text pada gambar

setiap gambar biasanya bila di beri effect tidak terlalu kelihatan untuk mempermudah untuk mengetahuinya bisa di lakukan dengan penambahan text pada gambar seperti berikut.

display(img)
text(x = 20, y = 280, label = "Ini Aji Bagas Prakasa", adj = c(0,1), col = "white", cex = 2)

Pada gambar di atas dapat di ketahui bahwa font yang muncul kurang jelas maka dari itu akan di beri semacam ourtline supaya lebih jelas font yang tampil seperti berikut.

display(img)
text(x = 20, y = 280, label = "Ini Aji Bagas Prakasa", adj = c(0,1), col = "black", cex = 2)
text(x = 19, y = 279, label = "Ini Aji Bagas Prakasa", adj = c(0,1), col = "white", cex = 2)

Nah setelah jadi seperti ini akan membuatnya semakin mudah untuk di baca.

Brightness

pada opsi ini akan mengatur tingkat kecerahan pada gambar yang kita panggil. dengan simbol +/-

setwd("D:/Media/iseng/intro yt 2")
library(EBImage)
Image1 <- Image + 0.2
Image2 <- Image - 0.2
par(mfrow= c(1,2))
plot(Image1)
plot(Image2)

Adjusting Contrast

di opsi ini gambar akan di ubah kontrasnya. hanya saja di sini menggunakan simbol sama dengan (=) secara default gambar bernilai 1.

setwd("D:/Media/iseng/intro yt 2")
library(EBImage)
Image3 <- Image * 0.5
Image4 <- Image * 2
par(mfrow= c(1,2))
plot(Image3)
plot(Image4)

Gamma Correction

pada opsi ini image akan di ubah gammanya. untuk caranya sama seperti kontras namun untuk gama menggunakan ^,

setwd("D:/Media/iseng/intro yt 2")
library(EBImage)
Image5 <- Image ^ 2
Image6 <- Image ^ 0.7
par(mfrow= c(1,2))
plot(Image5)
plot(Image6)

Spatial Transformation

untuk spatial tranformation image akan di setting rotasinya. sesuai value yang ada di Imagetr <- translate(rotate(Image, 45), c(50, 0))

setwd("D:/Media/iseng/intro yt 2")
library(EBImage)
Imagetr <- translate(rotate(Image, 45), c(50, 0))
display(Imagetr)

filtering

untuk filtering merupakan cara yang di bilang paling panjang dan berat untuk di jalankan. namun dengan filter akan berdampak sangat besar pada gambar. untuk hasil dan caranya seperti berikut.

setwd("D:/Media/iseng/intro yt 2")
library(EBImage)
fLow <- makeBrush(21, shape= 'disc', step=FALSE)^2
fLow <- fLow/sum(fLow)
Image.fLow <- filter2(Image, fLow)
display(Image.fLow)

fHigh <- matrix(1, nc = 3, nr = 3)
fHigh[2, 2] <- -8
Image.fHigh <- filter2(Image, fHigh)
display(Image.fHigh)

aa<- medianFilter(Image.fHigh,1.1)
par(mfrow= c(1,2))
plot(aa)
plot(Image.fHigh)

Image <- readImage("profil.jpg")
medFltr <- medianFilter(Image, 1.1)
display(medFltr)
